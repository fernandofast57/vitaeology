# PROMPT PER CLAUDE CON CANVA - 21 Thumbnail Challenge

Copia e incolla questo prompt nel tuo Claude con accesso a Canva:

## PROMPT

Devo creare 21 thumbnail per video challenge. Formato 9:16 verticale (1080×1920 px).

### STILE COMUNE A TUTTI:
- Sfondo: gradiente colorato (colore specifico per challenge)
- Volto di un uomo con occhiali (Fernando) sulla destra, sorridente
- Testo "GIORNO X" in alto, piccolo, uppercase, bianco
- Titolo grande al centro-sinistra, uppercase, bold, bianco
- Icona play button al centro (cerchio bianco con triangolo)
- Logo "Vitaeology" in basso a destra, piccolo, bianco
- Font: sans-serif bold (Montserrat o simile)
