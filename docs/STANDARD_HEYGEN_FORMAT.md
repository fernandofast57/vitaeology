## SCHEMA PRODUZIONE TABELLARE

Ogni video segue questo schema tabellare production-ready:

### STRUTTURA BASE

```
| Timing    | Scene         | Voiceover                              | Visual                                |
|-----------|---------------|----------------------------------------|---------------------------------------|
| 0-4s      | 1. [TITOLO]   | "Testo voiceover..."                   | A ROLL: [descrizione avatar]          |
|           |               |                                        | B ROLL: [se presente]                 |
|           |               |                                        | OVERLAY: [testo/grafica se presente]  |
```

---

## TIPI DI VISUAL

### A ROLL (Avatar)
Inquadratura principale con avatar Fernando.

| Tipo          | Descrizione                    | Quando Usare              |
|---------------|--------------------------------|---------------------------|
| `Close-up`    | Volto ravvicinato              | Hook, enfasi emotiva      |
| `Medium shot` | Busto (3/4)                    | Spiegazioni, contenuto    |
| `Medium-close`| Volto + spalle                 | Transizioni, punti chiave |

### B ROLL (Supporto)
Materiale supplementare che copre o affianca l'avatar.

| Tipo               | Descrizione                           | Esempio                           |
|--------------------|---------------------------------------|-----------------------------------|
| `Stock footage`    | Video di repertorio                   | Mani che scrivono, ufficio        |
| `Animated graphic` | Grafica animata                       | Icone, schemi, transizioni        |
| `Text slide`       | Slide con testo                       | Citazioni, statistiche            |
| `Split screen`     | Avatar + contenuto                    | Avatar + lista punti              |

### OVERLAY (Sovrimpressioni)
Elementi grafici sovrapposti al video.

| Tipo              | Formato                               | Uso                               |
|-------------------|---------------------------------------|-----------------------------------|
| `Text overlay`    | Titolo/sottotitolo su video           | Enfasi parole chiave              |
| `Lower third`     | Barra info in basso                   | Nome, titolo, CTA                 |
| `Icon overlay`    | Icona animata                         | Bullet point visivi               |
| `Progress bar`    | Barra progresso                       | Video multi-step                  |

## REGOLE TIMING

### Durata Scene per Tipo

| Tipo Scena    | Durata Min | Durata Max | Note                              |
|---------------|------------|------------|-----------------------------------|
| HOOK          | 3s         | 5s         | Impatto immediato                 |
| PAUSA         | 1s         | 3s         | Dopo domanda retorica             |
| CONTESTO      | 8s         | 15s        | Setup problema                    |
| STORIA        | 12s        | 25s        | Aneddoto personale                |
| INSIGHT       | 8s         | 15s        | Lezione chiave                    |
| ESERCIZIO     | 15s        | 25s        | Istruzioni chiare                 |
| CHIUSURA      | 8s         | 15s        | Saluto + anticipazione            |

### Calcolo Crediti HeyGen

```
Crediti = (Durata in secondi / 60) × 10
Esempio: 90s video = 15 credits
```

## CHECKLIST PRE-PRODUZIONE

### Contenuto

- [ ] Hook cattura attenzione in <4 secondi
- [ ] Storia personale con anno verificato (Anno - 1957 = Età Fernando)
- [ ] Esercizio ha 3 step massimo
- [ ] Chiusura anticipa il giorno successivo
- [ ] Linguaggio validante (AVERE, non ESSERE/FARE)

### Visual

- [ ] A ROLL variato (non 2 inquadrature uguali consecutive)
- [ ] B ROLL usato per spezzare monotonia
- [ ] OVERLAY keywords presenti per SEO/retention
- [ ] Logo Vitaeology visibile hook + chiusura

### Tecnico

- [ ] Timing totale verificato
- [ ] Orientation corretta (9:16 social, 16:9 web)
- [ ] Metadata compilati
- [ ] Crediti calcolati

## NOTE TECNICHE HEYGEN

### Avatar IV Settings
- **Engine:** Avatar IV (per dettaglio gesti e espressioni)
- **More Expressive:** ON per video emotivi
- **Prompt Refinement:** ON

### Expression Reference

| Expression     | Descrizione                    | Scene Type              |
|----------------|--------------------------------|-------------------------|
| `Engaging`     | Occhi vivaci, leggero sorriso  | Hook                    |
| `Thoughtful`   | Sguardo riflessivo             | Domande retoriche       |
| `Understanding`| Empatico, annuisce             | Contesto problema       |

| `Wise`         | Sicuro, calmo                  | Insight                 |
| `Encouraging`  | Sorriso, gesti aperti          | Esercizi                |
| `Warm`         | Amichevole, rilassato          | Chiusura                |

- Risoluzione: 1080p (Full HD)
- Formato: MP4 (H.264)
