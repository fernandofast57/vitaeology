# Sistema di Revisione Contenuti Challenge

## Metodologia: 3 Fasi Sequenziali

```
FASE 3 (Prima) → FASE 2 → FASE 1 (Ultima)
   AVERE      →  AGENCY  →  ITALIANO
```

---

## FASE 3: Principio AVERE (Validazione vs Deficit)

### Regola Fondamentale
L'utente POSSIEDE GIÀ tutte le capacità. Ha la LIBERTÀ di riconoscerle e usarle.

### Trasformazioni

| Pattern VIETATO | Trasformazione AVERE |
|-----------------|----------------------|
| "Ti manca..." | "Hai già... puoi scegliere di espanderlo" |
| "Non hai..." | "Hai... puoi attivarlo quando vuoi" |
| "Devi sviluppare..." | "Possiedi già... puoi riconoscerlo" |
| "Imparerai a..." | "Riconoscerai quello che già fai quando..." |
| "Acquisirai..." | "Vedrai quello che hai già..." |
| "Sei/Non sei..." | "Hai/Possiedi la capacità di..." |
| "Fai/Non fai..." | "Hai la possibilità di..." |

### Checklist FASE 3
- [ ] Nessun "ti manca" o "non hai" come sentenza
- [ ] Nessun "devi sviluppare/imparare/acquisire"
- [ ] Nessun giudizio ESSERE ("sei/non sei")
- [ ] Nessun giudizio FARE ("fai/non fai")
- [ ] Ogni capacità presentata come GIÀ POSSEDUTA
- [ ] Libertà di scelta sempre implicita

---

## FASE 2: Principio Agency (Facilitazione vs Prescrizione)

### Regola Fondamentale
L'utente è AGENTE ATTIVO, non paziente passivo. Mai prescrizioni dirette.

### Trasformazioni

| Pattern VIETATO | Trasformazione AGENCY |
|-----------------|----------------------|
| "Scrivi..." (imperativo) | "Potresti annotare..." / "Se vuoi, scrivi..." |
| "Fai..." (imperativo) | "Hai la possibilità di..." |
| "Devi..." | "Puoi scegliere di..." |
| "Prendi carta e penna" | "Con carta e penna, se preferisci..." |
| "Non fare X" | "Potresti lasciare da parte X" |
| "Ricorda che..." | "Vale la pena notare che..." |
| "Pensa a..." | "Potresti considerare..." |

### Strutture Preferite
- Domande aperte: "Cosa succederebbe se...?"
- Inviti: "Potresti esplorare..."
- Opzioni: "Una possibilità è..."
- Riconoscimenti: "Forse hai già notato..."

### Checklist FASE 2
- [ ] Nessun imperativo diretto (scrivi, fai, prendi, pensa)
- [ ] Nessun "devi" o "dovresti"
- [ ] Presenza di "potresti", "puoi scegliere", "hai la possibilità"
- [ ] Domande invece di comandi dove possibile
- [ ] L'utente mantiene sempre il controllo

## FASE 1: Sintassi Italiana Naturale

### Regola Fondamentale
Evitare costruzioni che suonano come traduzioni dall'inglese.

### Trasformazioni

| Pattern ANGLOFONO | Italiano Naturale |
|-------------------|-------------------|
| "È quello che farai" | "Questo è ciò che potrai fare" |
| "Quello che ti mostrerò" | "Ciò che vedremo insieme" |
| "Quando ti rendi conto che" | "Nel momento in cui noti che" |
| "Il fatto che tu senta" | "Il fatto stesso che tu senta" / "Che tu senta" |
| "C'è qualcosa che" | "Esiste qualcosa che" / "Qualcosa che" |
| "Questo è il regalo che ti sei fatto" | "Ecco il dono che ti sei concesso" |
| "La cosa più importante" | "L'aspetto più importante" / "Ciò che conta di più" |
| "Il problema è che" | "La questione è che" / "Il punto è che" |
| "Non è colpa tua" | "Non dipende da te" / "Non è responsabilità tua" |

### Gerundi (usare con cautela)
| Anglofono | Italiano |
|-----------|----------|
| "Stai cercando" (continuo) | "Cerchi" (presente) |
| "Stai facendo" | "Fai" |
| "Continuando a fare" | "Se continui a fare" / "Continuando con" |

### Strutture da evitare
- Frasi troppo lunghe con incisi multipli
- Soggetto lontano dal verbo
- Accumulo di subordinate
- "Quello/quella che" ripetuto

### Checklist FASE 1
- [ ] Frasi con soggetto vicino al verbo
- [ ] Nessun "quello che" ripetuto nella stessa frase
- [ ] Gerundi usati correttamente (non come in inglese)
- [ ] Ritmo italiano (frasi più brevi, punteggiatura frequente)
- [ ] Suona naturale letto ad alta voce

## PROCESSO DI REVISIONE

Per ogni giorno di challenge:

### Step 1: Copia il testo originale

### Step 2: FASE 3 - Applica principio AVERE
- Sottolinea ogni deficit/giudizio
- Trasforma in possesso/libertà
- Verifica checklist

### Step 3: FASE 2 - Applica principio AGENCY
- Sottolinea ogni imperativo/prescrizione
- Trasforma in invito/possibilità
- Verifica checklist

### Step 4: FASE 1 - Applica sintassi italiana
- Leggi ad alta voce
- Correggi costruzioni anglofone
- Spezza frasi troppo lunghe
- Verifica checklist

### Step 5: Revisione finale
- Rileggi tutto
- Verifica coerenza
- Conferma che il messaggio originale è preservato
